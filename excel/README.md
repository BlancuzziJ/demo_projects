# Excel Demos

Spreadsheet models, dashboards, and advanced formula examples built in Microsoft Excel.

## Projects

| Project | File | Description |
|---------|------|-------------|
| Budget Tracker | `budget_tracker.xlsx` | Personal monthly budget with dynamic charts, data validation dropdowns, and conditional formatting |

> **Note:** `.xlsx` files are binary and tracked in Git via [Git LFS](https://git-lfs.com/).  
> If you have Git LFS installed, the files are downloaded automatically on clone.  
> Otherwise, download them directly from the [Releases](../../releases) page.

## Budget Tracker Highlights

- **Income & Expense sheets** – Enter transactions with category, date, and amount
- **Dashboard sheet** – Auto-updating summary table and doughnut chart
- **Data Validation** – Category dropdowns to ensure consistent entries
- **Conditional Formatting** – Rows turn red when a category is over budget
- **Formulas used** – `SUMIF`, `IFERROR`, `VLOOKUP`, `TEXT`, named ranges

## Opening the Files

1. Download the `.xlsx` file.
2. Open in Microsoft Excel 2016+ or Google Sheets (most features are compatible).
3. Enable macros if prompted (no macros are used in this demo).
